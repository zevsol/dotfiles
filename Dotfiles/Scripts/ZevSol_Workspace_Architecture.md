\# ZevSol 工作区架构蓝图> \*\*版本:\*\* 1.0.0 | \*\*作者:\*\* Zev | \*\*状态:\*\* 生效中  > \*\*设计理念:\*\* 绝对隔离。零 C 盘污染。高性能执行。策略性冷备份。\---\## 💿 D:\\ \[ 工作核心 ]\*\*介质:\*\* NVMe 固态硬盘 | \*\*聚焦:\*\* 高频 I/O、编译速度、AI 执行```D:\\├── AI\_Workspace/           # 🧠 AI 重型资产（与 C 盘隔离）│   ├── Models/             # Ollama、HuggingFace 权重文件（通过环境变量拦截）│   └── Train\_Cache/        # 微调/训练用临时缓存├── Dotfiles/               # ⚙️ 版本控制的配置与脚本│   ├── scripts/            # 部署及自动化脚本│   ├── Terminal/           # Windows 终端设置（settings.json）│   ├── PowerShell/         # PowerShell Core 全局配置文件│   └── Config/             # 通用工具配置（Neovim、Git）├── Env/                    # 🚀 纯净开发基础环境│   ├── IDE\_Data/           # 包管理器及 IDE 缓存（Pip、Npm、JetBrains）│   ├── Scoop/              # Scoop 根目录（应用、持久化、快捷方式）│   └── WSL\_Distros/        # WSL2/Nix 虚拟硬盘文件（.vhdx）├── Media/                  # 🎨 高性能视觉素材│   ├── Brand\_Assets/       # ZevSol 品牌资源（Logo、字体、情绪板）│   ├── Photography/        # 需要快速磁盘读写的高分辨率/RAW 照片│   └── Projects\_Raw/       # 进行中的 UI/UX 草稿或视频项目└── Projects/               # 💻 核心代码仓库&#x20;   ├── Lab/                # 演示项目、爬虫、克隆的开源项目&#x20;   └── ZevSol/             # 核心个人品牌仓库及产品线```\---\## 💾 E:\\ \[ 保险库与恢复 ]\*\*介质:\*\* 固态硬盘 | \*\*聚焦:\*\* 冷存储、离线灾难恢复、高安全资产```E:\\├── Recovery/               # 🚑 灾难恢复工具包│   ├── Drivers/            # 离线驱动程序（主板、网卡、显卡）│   ├── Images/             # Windows 操作系统镜像及 PE 工具│   └── Standalone/         # 与 Scoop 不兼容的便携工具（如 Rufus）├── SoftStore/              # ☢️ 高权限软件隔离区│   └── System\_Level/       # 需要管理员权限或非便携安装包（VPN、杀毒软件）├── Temp/                   # 🗑️ 全局垃圾/缓冲区域│   └── Downloads/          # 浏览器默认下载目录（定期清理）└── Vault/                  # 🔐 \[最高安全级别] 数字资产&#x20;   ├── Backups/            # 冷备份数据库及压缩的遗留代码仓库&#x20;   └── Personal\_Photos/    # 按时间顺序整理的个人生活档案```\---\## 🛡️ 核心防污染环境变量| 变量名 | 目标重定向路径 | 用途 || :--- | :--- | :--- || `SCOOP` | `D:\\Env\\Scoop` | 所有便携式 CLI/GUI 应用根目录 || `OLLAMA\_MODELS` | `D:\\AI\_Workspace\\Models\\Ollama` | 拦截 Ollama 本地权重文件 || `HF\_HOME` | `D:\\AI\_Workspace\\Models\\HuggingFace` | 拦截 HuggingFace 缓存 || `PIP\_CACHE\_DIR` | `D:\\Env\\IDE\_Data\\Pip\_Cache` | 防止 pip 占用 AppData 空间 || `NPM\_CONFIG\_CACHE` | `D:\\Env\\IDE\_Data\\Npm\_Cache` | 防止 npm 占用 Roaming 空间 |\---\## ⚡ 部署脚本要在新机器上初始化或恢复此架构，请执行引导脚本：👉 \*\*\[运行 Bootstrap\_Clean\_Workspace.ps1](./scripts/Bootstrap\_Clean\_Workspace.ps1)\*\*
